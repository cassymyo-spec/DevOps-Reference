# DevOps Reference Materials

This repository is a **reference hub** for server virtualization, Docker container setup, and staging/production architecture.  

## Contents

- `docs/` - Diagrams, notes, and architecture explanations  
- `reference-configs/` - Example configuration files (Docker Compose, NGINX, Dockerfile)  
- `scripts/` - Optional reference scripts  
- `diagrams/` - Visual representations of staging vs production workflow  

## Purpose

This repo is for **learning, planning, and documentation purposes**, not for direct deployment. It helps maintain a clear understanding of DevOps setup and best practices.
